Home

SeismicFlow is an all-in-one seismic and well analysis application with built-in TensorFlow, PyTorch, and Scikit-learn functionalities. It allows seamless integration of machine learning and AI tools for advanced seismic analysis. From well log analysis to fault prediction, clustering, and 3D visualization, SeismicFlow simplifies complex workflows while providing cutting-edge capabilities.

About

SeismicFlow was created by Matin Mahzad, a geophysics PhD candidate, with the goal of integrating AI into seismic processing and interpretation workflows. Matin specializes in the application of machine learning to seismic data, transforming seismic interpretation by providing AI-driven tools for quantitative analysis.

Features

  • Facies Prediction Module: Neural network-based Facies segmentation, 2D/3D visualization, and GPU-accelerated inference.
  • Denoising Module: Neural network-based and non-neural network-based denoising, 2D/3D visualization, and GPU-accelerated inference.
  • Fault Prediction Module: Neural network-based fault segmentation, 2D/3D visualization, and GPU-accelerated inference.
  • Well Log Analysis: Crossplots, regression, clustering (13 algorithms), and categorization of well logs with traditional and neural network models.
  • Petrophysical Characterization: Integration of well log and seismic data, 3D petrophysical property predictions, and facies categorization.
  • Customizable Workflows: Design your own neural network models, apply Bayesian optimization for hyperparameter tuning, and visualize results in 3D.
  • Clustering: Advanced clustering techniques, including self-organizing maps (SOM) and deep embedded clustering (DEC).
  • Multi-Language Capable: The application works with C++, Python, and JavaScript, allowing you to develop and add plugins in any of these languages.
  • Plugin Development: Create your plugins with C++, Python, or JavaScript and continue building on this application.
  • Inter-Plugin Communication: Plugins can communicate with the main application using methods such as sockets and server-client protocols.
